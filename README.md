# React Native Ignite Gym

This is a React Native app for saving your workout routines at the gym.


### Setup Environment

Project created with Expo.[ See official docs](https://docs.expo.dev/)

### Run

**Install dependencies**

```
yarn
```
a
**Run IOS**

```
yarn ios
```

**Run Android**

```
yarn android
```
